☕ Coffee Orders Sales Dashboard & Data Analysis
📌 Project Description

The Coffee Orders Sales Dashboard project focuses on analyzing coffee shop sales data to understand customer purchasing behavior, product popularity, and sales trends.

Using Microsoft Excel, the dataset was cleaned, processed, and analyzed to create an interactive dashboard that provides meaningful insights for business decision-making.

The goal of this project is to help coffee shop owners:

Understand customer preferences

Identify peak sales hours

Optimize inventory management

Improve marketing strategies

📊 Dashboard Features

The dashboard provides the following key analytics:

☕ Product Performance

Most popular coffee drinks

Least ordered beverages

Sales distribution by product category

📦 Order Details Analysis

Best-selling drink sizes

Popular add-ons and customizations

Number of orders by category

⏰ Time-Based Insights

Peak ordering hours

Daily and weekly sales trends

Time-of-day purchase patterns

💰 Revenue Insights

Total revenue generated

Revenue contribution by product

High-value product identification

📊 Data Analysis Workflow

The project follows a structured data analysis pipeline:

1️⃣ Data Collection

Import raw coffee order dataset.

2️⃣ Data Cleaning

Remove duplicate entries

Handle missing values

Standardize product names and categories

3️⃣ Data Transformation

Create calculated columns

Format date and time fields

Prepare data for analysis

4️⃣ Data Analysis

Use pivot tables to analyze sales patterns

Identify top-selling products

Analyze customer purchase behavior

5️⃣ Dashboard Creation

Build an interactive dashboard

Use charts and slicers for filtering insights

📈 Key Insights

Some of the insights derived from the analysis include:

✔ Most popular coffee drink ordered by customers
✔ Peak ordering hours during the day
✔ Best-selling drink sizes
✔ Products with the highest revenue contribution
✔ Low-performing items that may need removal or promotion
